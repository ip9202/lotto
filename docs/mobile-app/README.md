## 🎯 문서 사용 가이드

### **A. 개발 시작 전**
1. **PRD 문서** 읽기 - 제품 요구사항 파악
2. **개발 계획서** 확인 - 단계별 개발 계획
3. **디자인 가이드** 검토 - UI/UX 가이드라인

### **B. 개발 진행 중**
1. **개발 계획서** 체크리스트 활용
2. **디자인 가이드** 참고하여 UI 구현
3. **기술 문서** 참고하여 아키텍처 구현

### **C. 개발 완료 후**
1. **배포 가이드** 따라 앱 배포
2. **앱스토어 가이드** 따라 등록
3. **유지보수 가이드** 따라 운영
